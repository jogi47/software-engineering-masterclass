# DSA Problems

This folder contains TypeScript implementations for data structures, algorithms, and interview-style coding problems.

## What Is Here

| Section | Count | Purpose |
| --- | --- | --- |
| `core/` | 12 | Foundational data structures and sorting algorithms |
| `easy/` | 18 | Introductory LeetCode-style problems |
| `medium/` | 47 | Core interview-prep problems |
| `hard/` | 11 | Advanced problems and techniques |

## How To Read This Folder

- Start with `core/` if you want to build the underlying data-structure intuition first.
- Move to `easy/` for pattern recognition and basic implementation fluency.
- Spend most interview prep time in `medium/`.
- Use `hard/` to push on advanced techniques like heaps, monotonic data structures, advanced binary search, and backtracking.

## Run Files

```bash
npx ts-node 02-dsa-problems/core/001-dynamic-array.ts
npx ts-node 02-dsa-problems/easy/001-two-sum.ts
npx ts-node 02-dsa-problems/medium/001-group-anagrams.ts
npx ts-node 02-dsa-problems/hard/001-trapping-rain-water.ts
```
