# Database Basics

This folder contains database and distributed-data notes. Right now it is centered on chapter-by-chapter notes from *Designing Data-Intensive Applications* (DDIA).

## What Is Here

| Section | Count | Purpose |
| --- | --- | --- |
| `ddia/foundations/` | 4 | Core properties of data systems, models, storage, and evolution |
| `ddia/distributed-data/` | 5 | Replication, partitioning, transactions, and consensus |
| `ddia/derived-data/` | 3 | Batch and stream processing, plus future-looking system design ideas |

## How To Read This Folder

- Read `foundations/` first if you are new to database internals.
- Use `distributed-data/` when preparing for system design interviews or backend architecture work.
- Read `derived-data/` once you are comfortable with storage and distributed-system basics.
